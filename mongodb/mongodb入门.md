# mongodb入门文档 #
  1.首先到官网下载，安装MongoDB（注意MongoDB还有一个可视化管理工具叫： MongoVUE ）
  2.自己手动在任意盘符下建一个 MongoDB的文件夹 目录，并将下载的 MongoDB 解压，然后把 解压后的bin目录，拷贝到MongoDB文件夹下。
  新建一个 ： 数据库文件夹（ data ）
  新建一个 ： 日志文件夹  ( log )
